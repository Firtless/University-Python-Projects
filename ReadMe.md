## Умова:

Дано шахівницю, знайдіть найкоротшу відстань (мінімальна кількість ходів),
яку пройшов кінь, щоб досягти заданого пункту призначення з заданого
джерела.

Результат: мінімальна необхідна кількість кроків – 6

Результуючий файл має містити кількість ходів
Результуючий файл output.txt має містити значення найкоротшого шляху від
початкової точки до точки призначення

## Творче:

Реалізація системи автономної навігації для складського робота, яка використовує алгоритм BFS для пошуку найкоротшого маршруту в обхід перешкод.

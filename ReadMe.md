## Умова:

Завдання: Напишіть функцію, яка перевіряє, чи є заданий масив монотонним. Функція
повинна повертати логічне значення True, якщо масив є монотонним, і False, якщо
масив не є монотонним.

Додаткове завдання: доповнити програму для того щоб воно порівнювало масив з температур і повідомляло про стабільність або зміни в температурі в довгому списку введення даних
